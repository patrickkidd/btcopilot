**Rule clarifications from Meeting 6 (not new rules, refinements of existing):**

- **Rule 20 (baseline vs. shift):** A single statement can yield two entries — one for a baseline and one for an intensification at a later timestamped event. Sylvia's sensitivity: code (1) baseline anxiety up, unknown date; (2) anxiety up at Claire's birth (shift from episodic to permanent). These are distinct time points, not duplication.
- **Shift-only coding:** Only code the shift, not the prior established pattern. If early statements describe an existing pattern and later statements show the shift from it, code only the shift. The background pattern is coded once when first established; re-descriptions of it are reiterations.
