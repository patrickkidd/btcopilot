Statement Extraction Progress Demo

Discussion Details

This demo shows the extraction progress bar for a discussion.

Discussion ID: 123

How it works:

  1. The progress bar polls /therapist/discussions/{id}/progress every 2 seconds
  2. It shows the number of statements processed out of the total
  3. The progress bar automatically hides when all statements are processed
  4. If extraction stalls, it can trigger the background job to continue

Integration Example:

// Initialize extraction progress for a discussion
window.extractionProgress.init(discussionId);

// Or specify a custom container
window.extractionProgress.init(discussionId, 'my-container-id');

// Clean up when done
window.extractionProgress.destroy();
